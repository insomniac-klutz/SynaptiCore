# SynaptiCore

Genesis Playground <> Autonomous Agents

## Overview

SynaptiCore is a Python library that provides tools and frameworks for building and deploying autonomous AI agents. The library integrates with various language models and frameworks like LangChain, LangGraph, and SmolAgents to create powerful AI applications.

## Features

- **Apps** - Ready-to-use applications including DeCypher
- **Core** - Core functionality and utilities
  - **a2aPro** - Agent-to-agent communication framework
  - **mcPro** - Model Context Protocol implementations
- **Servers** - Server implementations for various protocols
- **Tools** - Utility tools for language models and more

## Installation

```bash
pip install synapticore
```
